# sales-prediction
📈 Sales Prediction using Machine Learning
📌 Overview

This project predicts product sales based on advertising budgets spent on different platforms such as TV, Radio, and Newspaper. It uses Machine Learning techniques to analyze advertising data and forecast future sales.

🎯 Objectives
Predict sales using advertising expenditure data.
Perform data cleaning and preprocessing.
Train and evaluate a Machine Learning model.
Analyze the impact of advertising on sales.
Build an interactive web application using Streamlit.
🛠️ Technologies Used
Python
Pandas
NumPy
Scikit-learn
Matplotlib
Streamlit
Joblib
📊 Dataset Features
TV Advertising Budget
Radio Advertising Budget
Newspaper Advertising Budget
Sales (Target Variable)
🤖 Model Used
Linear Regression
📈 Result
Achieved high prediction accuracy using Linear Regression.
Identified that TV and Radio advertisements have a stronger influence on sales compared to Newspaper advertisements.
🚀 How to Run
pip install -r requirements.txt
python src/train.py
python src/predict.py
python -m streamlit run app.py
💡 Outcome

This project demonstrates how Machine Learning can be used to predict sales and help businesses make better marketing decisions by understanding the relationship between advertising expenditure and sales performance.
